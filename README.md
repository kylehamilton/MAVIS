MAVIS: Meta Analysis via Shiny
=====

The goal of this project is to help students and researchers run a meta-analysis as easily as possible.

List of Packages Used 
```
library(shiny) 
library(shinyAce) 
library(metafor) 
library(meta) 
library(MAd) 
library(MAc) 
library(quantreg) 
library(ggplot2) 
```
